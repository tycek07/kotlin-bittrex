# kotlin-bittrex
Unofficial kotlin api library for Bittrex 


Add to your project using gradle:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  
  ```
  	dependencies {
	        implementation 'com.github.tycek07:kotlin-bittrex:master-SNAPSHOT'
	}
  ```
